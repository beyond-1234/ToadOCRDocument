## 一、Install ECTD

### 1、Ubnutu\Debian

```sh
$ sudo apt-get install etcd
```

### 2、CentOS

```sh
$ yum install etcd
```

### 3、检查版本

```sh
$ etcd --version
```

输出

```

Git SHA: your git version
Go Version: your go verson
Go OS/Arch: linux/amd64
```
